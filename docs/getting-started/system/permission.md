# 权限名称

## 新增权限

“新增权限”功能允许系统管理员或具有相应权限的管理员，为已存在的用户追加新的系统操作权限。
此操作通常用于,对用户调用API鉴权。

点击添加权限，吊起权限表单
![add_permission](/assets/img/system/permission/add_permission.png)

点击保存添加权限,添加权限表单

![add_permission_form](/assets/img/system/permission/add_permission_form.png)
